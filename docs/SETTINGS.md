# Settings

Taskboard settings are stored as user preferences and control interface behavior without changing task data.

## Current settings

- Theme: dark, light or system
- Accent color mode
- UI language foundation: German or English
- Week start
- Default view
- Completion sound
- Delete sound
- Delete confirmation
- Task count visibility
- Notification permission preparation

## Design goal

Settings keep the board adaptable while preserving a simple daily workflow. Most options affect presentation, feedback or safety rather than the underlying data model.

## Future direction

A larger design system could replace some styling with semantic tokens and shared CSS variables.
