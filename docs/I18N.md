# Internationalisierung (Deutsch / Englisch)

Die App nutzt eine einfache lokale Übersetzungsschicht in `lib/i18n.ts`.

Wichtig:

- Es werden nur UI-Texte übersetzt.
- Board-, Listen-, Aufgaben-, Notiz- und Label-Inhalte werden niemals automatisch übersetzt.
- Die gewählte Sprache wird in den lokalen App-Einstellungen gespeichert.
- Komponenten nutzen `useI18n()` und rufen Texte über `t("key")` ab.

Beispiel:

```tsx
const { t } = useI18n();

<button>{t("board.addList")}</button>
```

## Woche beginnt am

Die Einstellung `weekStartsOn` unterstützt jetzt alle sieben Wochentage:

- Montag
- Dienstag
- Mittwoch
- Donnerstag
- Freitag
- Samstag
- Sonntag

Aktuell wird die Einstellung gespeichert und für spätere Kalender-/Wochenlogik vorbereitet. Relevant wird sie besonders für kommende Features wie Wochenfilter, Wiederholungen und Wochenansichten.
